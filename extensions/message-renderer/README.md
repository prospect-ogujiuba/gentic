# message-renderer

Custom message renderers.

## Discovery

Extensions live at `extensions/message-renderer/<name>/index.ts`. Put helper files beside that `index.ts`.

## Example Gentic extensions

- **Plan Renderer** — renders Gentic planning messages as structured TUI blocks.
- **Audit Renderer** — displays audit entries compactly in the transcript.
- **Tool Summary Renderer** — formats verbose tool results into readable sections.
