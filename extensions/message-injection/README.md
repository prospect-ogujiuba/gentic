# message-injection

Custom, steering, follow-up, and user message injection.

## Discovery

Extensions live at `extensions/message-injection/<name>/index.ts`. Put helper files beside that `index.ts`.

## Example Gentic extensions

- **Follow-up Nudger** — queues a follow-up user message after a workflow completes.
- **Steering Message** — injects real-time guidance while the agent is active.
- **Session Note** — stores a visible custom message with Gentic workflow state.
