# shortcut

Keyboard shortcut extensions.

## Discovery

Extensions live at `extensions/shortcut/<name>/index.ts`. Put helper files beside that `index.ts`.

## Example Gentic extensions

- **Plan Toggle** — binds a key to toggle planning mode.
- **Checkpoint Shortcut** — labels the active session entry from the keyboard.
- **Tool Mode Shortcut** — cycles between strict and permissive tool policies.
