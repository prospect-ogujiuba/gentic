# ui-component

Custom TUI components and UI interactions.

## Discovery

Extensions live at `extensions/ui-component/<name>/index.ts`. Put helper files beside that `index.ts`.

## Example Gentic extensions

- **Review Picker** — presents an interactive selector for review targets.
- **Workflow Dashboard** — shows Gentic workflow state above the editor.
- **Risk Dialog** — asks for confirmation with rich action details.
