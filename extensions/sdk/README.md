# sdk

SDK integrations and programmatic pi usage.

## Discovery

Extensions live at `extensions/sdk/<name>/index.ts`. Put helper files beside that `index.ts`.

## Example Gentic extensions

- **Batch Runner** — uses pi programmatically to run Gentic workflows outside the TUI.
- **Integration Harness** — exposes Gentic operations to another Node application.
- **Evaluation Driver** — runs repeatable prompts against Gentic surfaces.
