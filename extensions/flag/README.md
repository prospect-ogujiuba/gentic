# flag

CLI flag registration and handling.

## Discovery

Extensions live at `extensions/flag/<name>/index.ts`. Put helper files beside that `index.ts`.

## Example Gentic extensions

- **Plan Flag** — starts pi with Gentic planning behavior enabled.
- **Dry Run Flag** — makes Gentic tools preview actions instead of applying them.
- **Profile Flag** — selects a Gentic behavior profile at startup.
