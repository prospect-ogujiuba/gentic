# model

Model catalog and metadata surfaces.

## Discovery

Extensions live at `extensions/model/<name>/index.ts`. Put helper files beside that `index.ts`.

## Example Gentic extensions

- **Model Catalog** — documents Gentic-supported model traits and defaults.
- **Capability Tags** — maps models to tool-use, vision, or reasoning capabilities.
- **Routing Notes** — records preferred models for specific Gentic workflows.
