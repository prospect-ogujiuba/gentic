# command

Slash command extensions.

## Discovery

Extensions live at `extensions/command/<name>/index.ts`. Put helper files beside that `index.ts`.

## Example Gentic extensions

- **Plan Command** — adds a slash command for creating or updating an implementation plan.
- **Review Command** — summarizes current session changes and asks the model for targeted review.
- **Checkpoint Command** — labels the current session state before a risky refactor.
