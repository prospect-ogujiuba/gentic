# session-labels

Session entry labels and navigation markers.

## Discovery

Extensions live at `extensions/session-labels/<name>/index.ts`. Put helper files beside that `index.ts`.

## Example Gentic extensions

- **Auto Labeler** — labels important session entries after milestones.
- **Release Marker** — marks entries that correspond to release checkpoints.
- **Review Bookmark** — attaches review-oriented labels for tree navigation.
