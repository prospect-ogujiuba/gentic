# model-control

Model selection and thinking-level control.

## Discovery

Extensions live at `extensions/model-control/<name>/index.ts`. Put helper files beside that `index.ts`.

## Example Gentic extensions

- **Thinking Preset** — adjusts thinking level from model selection events.
- **Model Guard** — prevents switching to models that lack required capabilities.
- **Cost Mode** — selects cheaper models for low-risk tasks.
