# theme

Theme JSON files exposed by the Gentic suite.

## Discovery

Use `themes/<category>/<name>.json`.

## Example Gentic resources

- **Gentic Dark** — a dark theme tuned for long coding sessions.
- **Gentic Mono** — a low-color theme for distraction-free terminals.
- **Gentic High Contrast** — a theme optimized for accessibility and clear status colors.
